PMCC Poor Man's Covered Call · 穷人的备兑 Long Diagonal

用深度 ITM LEAPS Call替代 100 股正股，再卖近月 OTM Call 收权利金。资金效率比传统 Covered Call 高 4-5 倍。本质上是 Diagonal Spread 的特化形式。

构成
买入Call深度 ITM (Δ ≈ 0.80-0.90) · LEAPS 1+ 年
卖出CallOTM (Δ ≈ 0.30) · 30-45 天
资金占用 ≈ Long Call 权利金 (≈ 30% 正股价) 最大风险 ≈ Long Call 权利金 - 累计 Short Call 权利金 月度收益 ≈ Short Call 权利金 / Long Call 成本 (~3-8%)
Greeks 特征
Delta+0.50 ~ +0.70（Long 0.85 - Short 0.30） Theta正（Short 腿 Theta > Long 腿 Theta） Vega轻微正（Long 远期 ν 大于 Short 近期）
适用场景：看好长期标的但资金有限、想用杠杆做 Covered Call 风格的现金流策略；适合大市值蓝筹（AAPL/MSFT）。
✓ 优点
  • 资金占用是正股版 Covered Call 的 20-30%
  • 下行风险有限（最多损失 LEAPS 成本）
  • 每月可滚动 Short Call 收权利金
✗ 缺点
  • Long Call 没有正股的股息收益
  • Short Call 被行权时盈利上限较低
  • 需要每年滚动一次 LEAPS（Roll Out）

Calendar Spread Time Spread · 跨期价差 做多 Vega · Theta 收割

同一行权价，卖近月 + 买远月。利用近月 Theta 衰减比远月快的特性。本质上是做多波动率 + 做多时间的组合。

构成 (Call 版)
卖出Call行权价 K · 近月 (30 天)
买入Call行权价 K · 远月 (60-90 天)
净成本 = 远月 Call 权利金 - 近月 Call 权利金 (借方) 最大盈利 ≈ 近月到期时 S = K（理想情况） 最大亏损 = 净成本（标的远离 K 时） 盈利甜区 = K ± (远月 IV × √T_远月)
Greeks 特征
Delta~ 0（K 选择 ATM 时） Theta正（近月 Theta 衰减更快） Vega正（远月 Vega 大于近月） Gamma负（短端 Γ 大于长端）
适用场景：预期标的横盘到近月到期，且 IV 偏低（预期 IV 上升）。常用于财报前 - 不押方向但赌 IV 升高。
✓ 优点
  • 净成本低，风险有限
  • 同时受益于 Theta 衰减 + Vega 增长
  • 对方向判断要求低（K 选 ATM 即可）
✗ 缺点
  • 标的大幅偏离 K 时全损
  • 需要近月 IV 在到期前不暴跌
  • Skew 影响：远期 IV 通常 < 近期 IV（不利）

Diagonal Spread Diagonal · 对角价差 跨期 + 跨价

同时跨期（不同到期）和跨价（不同 K）的混合策略。Calendar 的方向化版本——可以加入方向偏好。PMCC 是 Diagonal 的特化（远月深度 ITM + 近月 OTM）。

构成 (看涨 Diagonal)
卖出Call高 K · 近月
买入Call低 K · 远月
净成本 = Long Call(远月,低K) - Short Call(近月,高K) 通常为借方 (Debit) 最大盈利场景：近月到期时 S 略低于 Short K 最大风险 ≈ 净成本
适用场景：温和看涨 + 想做月度收入；或看好但有压力位的中长期持仓。比 Calendar 多了方向偏好，比 Bull Call Spread 多了月度滚动潜力。
✓ 优点
  • 融合方向 + Theta + Vega 三种盈利源
  • 近月到期后可继续滚动（"Diagonal Roll"）
  • 资金效率高于直接买远月 Long Call
✗ 缺点
  • 构造复杂，需要平衡 K 与 T 选择
  • 近月被指派时处理较麻烦
  • 对 Skew 敏感

Jade Lizard 翡翠蜥蜴 · 无上行风险 高 IV · 中性偏多

巧妙构造：Short Put + Bear Call Spread。设计目标是"无上行风险"——只要总收入 ≥ Call 价差宽度，标的怎么涨都不亏。仅下行有风险（Short Put 被指派）。Tom Sosnoff 推广的明星策略。

构成
卖出PutOTM Put (K_p)
卖出CallOTM Call (K_cs，靠近现价)
买入Call更 OTM Call (K_cl)
总收入 = Put 权利金 + Call 价差净收入 设计准则：总收入 ≥ K_cl - K_cs (即 Call 价差宽度) → 上行无风险（无论 S 涨多高都不亏） 下行最大亏损 = K_p - 总收入 (S → 0 时)
Greeks 特征
Delta+ (轻微正向偏多) Theta正（3 个 Short 腿 + 1 个 Long） Vega负（IV 下降有利）
适用场景：高 IV 环境下中性偏多；预期标的不会大跌。比 Short Strangle 安全（上行有限保护），比纯 Short Put 更激进（多收 Call 价差权利金）。
✓ 优点
  • 上行完全无风险（设计的精妙之处）
  • 三个 Short 腿，Theta 收益丰厚
  • 胜率高（标的不大跌即赚）
✗ 缺点
  • 下行风险类似 Short Put（K_p 高时风险大）
  • 构造需要精确计算（总收入 vs Call 价差宽）
  • 三腿手续费高

Ratio Spread 比例价差 · 1×2 / 1×3 不对称腿 · 高级

买 1 腿 + 卖 2 腿（或 1×3）的不对称价差。可设计为净收入或净支出。如果在 Short 侧附近到期，利润最大；但远端有无限亏损风险（裸卖部分）。

构成 (Call 1×2 比例)
买入CallK₁ (低) · 1 张
卖出CallK₂ (高) · 2 张
净权利金 = C(K₁) - 2·C(K₂) 通常为净收入 (Credit) 或接近零成本 最大盈利 = K₂ - K₁ + 净收入 (S = K₂ 时) 最大亏损 = 无限 (S → ∞) 上行平衡 = K₂ + (K₂ - K₁) + 净收入
Greeks 特征
Delta视构造为正或负 Theta通常正（净 Short 腿） Vega负（IV 下降有利） Gamma远端为负（裸卖暴露）
适用场景：精准预期到期价落在 K₂ 附近 + 高 IV 环境想收 Theta。是 Butterfly 的"无翼"版——盈亏比更高但远端有无限风险。
✓ 优点
  • 构造可零成本或净收入
  • K₂ 附近最大盈利可观
  • Theta 衰减带来稳定收益
✗ 缺点
  • 远端无限亏损风险（裸卖部分）
  • 保证金占用大
  • 新手禁忌策略
📌 学习路径建议：新手优先掌握 组合策略页 的 6 种基础。基础熟练后，按这个顺序进入进阶：PMCC（最直观、风险有限） → Calendar Spread（理解 Theta/Vega 跨期） → Diagonal（PMCC + Calendar 混合） → Jade Lizard（设计巧妙、风险可控） → Ratio Spread（最危险，需要充分理解保证金）。
📌 风险提示：除 PMCC 外，本页其他策略对 IV / Theta / Skew 敏感度都很高，纸面盈利常因 Skew 不利而打折。建议先用模拟账户跑 3 个月再用实仓。
