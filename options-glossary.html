关于本词典

本页系统性整理仓库内三个交互式页面（单腿策略 / 组合策略 / 策略指南）所涉及的全部期权术语。每个术语包含 含义解释、计算公式 与 影响因素。希腊字母部分附直观 SVG 曲线图，方便理解形状随股价的变化规律。

📐 公式约定：S 标的现价，K 行权价，T 距到期年化时间，r 无风险利率，σ 波动率。

1 五大希腊字母 (Greeks)

希腊字母衡量期权价格对各风险因子的敏感度，是期权定价与风险管理的核心。前四个使用频率最高。

Δ Delta — 价格敏感度
一阶 · 方向
含义：标的价格 S 每变动 1 元，期权价格变动多少。也可解读为期权到期实值 (ITM) 的近似概率。
取值范围：Long Call ∈ [0, +1]，Long Put ∈ [−1, 0]，Short 取相反符号。
Δ_call = N(d₁) Δ_put = N(d₁) − 1 = −N(−d₁)
影响因素
因素变化对 |Δ| 影响说明
价值状态OTM → ATM → ITM↑越深度 ITM，Δ 越接近 ±1（趋同标的）
到期时间 T↓ITM ↑ · OTM ↓临到期 ITM 期权 Δ 趋近 ±1，OTM 趋近 0
波动率 σ↑趋向 0.5高波动让 OTM 也有"机会"，Δ 向 0.5 靠拢
📌 组合 Delta 可加和：策略总 Δ = 各腿 Δ × 数量之和。Delta 中性（Σ Δ = 0）即不押方向，常用于做空波动率策略。
Γ Gamma — Delta 的变化率
二阶 · 加速度
含义：标的每变动 1 元，Delta 变化多少。是 Delta 的"加速度"，反映 Delta 对冲的难度。
形状：钟形曲线，在 ATM 附近最大；远离行权价时迅速衰减为 0。Long 期权 Γ > 0，Short 期权 Γ < 0。
Γ = N′(d₁) / (S · σ · √T) (Call/Put 相同)
影响因素
因素变化对 Γ 影响说明
价值状态ATM最大Δ 在 ATM 变化最快
到期时间 T↓ATM ↑↑ · 远离 K ↓临到期 ATM Γ 爆炸，OTM Γ 趋零
波动率 σ↑↓高波动使 Γ 钟形变扁平、向两端铺开
⚠ Gamma 风险：Short Straddle / Iron Condor 等卖方策略 Γ < 0，临近到期 Γ 急剧放大，标的大幅跳空会瞬间亏损。这就是"Gamma Risk"。
Θ Theta — 时间衰减
一阶 · 时间
含义：每过一天（其他不变），期权价格变动多少。Long 期权 Θ < 0（持有者每天亏权利金），Short 期权 Θ > 0（卖方每天赚时间价值）。
形状：ATM 时 Θ 最负，远离 K 时 Θ 趋零。
Θ_call = [−S·N′(d₁)·σ / (2√T)] − r·K·e^(−rT)·N(d₂) Θ_put = [−S·N′(d₁)·σ / (2√T)] + r·K·e^(−rT)·N(−d₂) (常除以 365 转化为"每日"衰减)
影响因素
因素变化对 |Θ| 影响说明
到期时间 T↓↑↑临到期 |Θ| 急剧放大（最后 30 天最猛）
价值状态ATM最大ATM 时间价值最厚，每天衰减最多
波动率 σ↑↑更高的时间价值意味更大的衰减绝对值
📌 每日衰减率不均匀：远期期权 Θ 接近线性；进入最后 30 天后呈√T 加速衰减。这是"Theta 收益策略"（如 Iron Condor）选月度到期的核心原因。
ν Vega — 波动率敏感度
一阶 · 波动率
含义：隐含波动率 IV 每上升 1 个百分点（如 25% → 26%），期权价格变动多少。Long 期权 ν > 0，Short 期权 ν < 0。
形状：钟形（比 Γ 更平缓），ATM 最大；远期期权 ν 大于近期。
ν = S · N′(d₁) · √T (Call/Put 相同, 通常除以 100)
影响因素
因素变化对 ν 影响说明
到期时间 T↑↑远期 ν 大（与 √T 成正比）
价值状态ATM最大OTM/ITM 越深，ν 越小
波动率 σ↑近似不变ν 是一阶导，对 σ 自身不敏感
⚠ Vol Crush 陷阱：财报前 IV 飙高，事件揭晓后 IV 暴跌（哪怕方向看对，期权也大幅贬值）。Long Straddle 财报后买入是经典送钱场景。→ 详见 Vol Crush
ρ Rho — 利率敏感度
一阶 · 利率
含义：无风险利率 r 每上升 1 个百分点，期权价格变动多少。Call ρ > 0（持有 Call 等价于推迟付钱，利率上升对持有者有利），Put ρ < 0。
实务：短期期权 ρ 极小，常被忽略；只在长期 LEAPS（≥ 1 年）或利率剧烈变动时才重要。
ρ_call = K · T · e^(−rT) · N(d₂) ρ_put = −K · T · e^(−rT) · N(−d₂) (通常除以 100)
影响因素
因素变化对 |ρ| 影响说明
到期时间 T↑↑↑ρ 与 T 近似线性，最敏感的因子
价值状态ITM↑深度 ITM 期权 |ρ| 最大
📌 短期月度交易者通常无需关注 ρ。但在加息/降息周期或交易长期权时（LEAPS），ρ 是仅次于 ν 的风险源。

2 Black-Scholes 定价模型

Fischer Black、Myron Scholes 与 Robert Merton 1973 年提出的欧式期权定价模型。本仓库三个交互页面的所有定价均基于此模型实现。

Black-Scholes 定价公式 — Pricing Formula
欧式 Call/Put 在无股息资产上的解析解。模型假设：股价服从几何布朗运动、无套利、无交易成本、利率与波动率为常数、可连续对冲。
C = S · N(d₁) − K · e^(−rT) · N(d₂) P = K · e^(−rT) · N(−d₂) − S · N(−d₁)
参数：S 现价 · K 行权价 · T 距到期年化时间（如 30 天 = 30/365）· r 无风险利率 · σ 波动率。
📐 实现见 bsPrice()，三个页面共用同一份 BS 内核。
d₁ d₂ 中间变量 — BS Intermediate
BS 公式中的两个中间表达式。本质：将股价与行权价的对数偏离用波动率单位标准化，再加上漂移修正。
d₁ = [ln(S/K) + (r + σ²/2)·T] / (σ·√T) d₂ = d₁ − σ·√T
金融含义：N(d₂) 是风险中性测度下到期时 ITM 的概率（行权概率）；N(d₁) 是更复杂的风险中性概率（包含支付股票的部分）。日常使用中，Δ_call ≈ N(d₁) 也常被解读为到期 ITM 的近似概率。
N(x) — 标准正态分布累积函数 (CDF)
标准正态分布在 (−∞, x] 区间的概率。N(0) = 0.5，N(+∞) = 1，N(−∞) = 0。
N(x) = (1/√(2π)) · ∫_(−∞)^x exp(−t²/2) dt
数值实现：JavaScript/Python 没有内置 CDF，本仓库用 erf 多项式近似（Abramowitz & Stegun 7.1.26）：N(x) = 0.5·(1 + erf(x/√2))。
N′(x) 或 φ(x) — 标准正态概率密度 (PDF)
标准正态分布的密度，等于 N(x) 的导数。Gamma、Vega、Theta 公式中都出现 N′(d₁)。
N′(x) = φ(x) = (1/√(2π)) · exp(−x²/2)
e^(−rT) — 折现因子 / Discount Factor
含义：把到期时刻的金额折算回今天的现值。BS 公式中 K·e^(−rT) 即"行权价的现值"。
影响：r 越大或 T 越长，折现得越多（系数越小）。是 Rho 的来源。
PV(K) = K · e^(−rT) 若 r = 5%, T = 1 年：折现因子 = e^(−0.05) ≈ 0.9512
Put-Call 平价 — Put-Call Parity
欧式 Call 与 Put 在同行权价同到期下的等式约束。这是模型无关的无套利关系，不依赖 BS 假设。
C − P = S − K · e^(−rT)
📌 应用：合成正股 = Long Call + Short Put @同 K；Long Call = 持股 + Long Put + 现金 K·e^(−rT)。所有合成头寸都源自这个等式。

3 期权基础术语

Call / Put — 看涨期权 / 看跌期权
Call (看涨)：买方获得在 T 时刻以 K 价买入标的的权利。
Put (看跌)：买方获得在 T 时刻以 K 价卖出标的的权利。
买方付权利金购买"权利"；卖方收权利金承担"义务"。
S 现价 — Spot Price / Underlying Price
标的资产当前的市场价格。BS 模型核心输入。期权价值随 S 实时变动（敏感度由 Delta 衡量）。
K 行权价 — Strike Price / Exercise Price
合约规定的行权约定价。买方有权（或义务，取决于 Call/Put 与多空方向）以此价格买卖标的。一份合约 K 在签订时锁定，不会随市场变动。
T 到期日 — Maturity / Expiration
期权失效的日期。BS 公式中以"年化时间"输入：T = (剩余天数) / 365。距到期越近，时间价值衰减越快（Theta 加速）。
📅 美股月度期权一般在每月第三个周五到期；A 股 50ETF 期权在每月第四个周三到期。
r 无风险利率 — Risk-Free Rate
理论上是无信用风险的连续复利收益率。常用代理：美国国债收益率（短期用 1M / 3M 国债，长期匹配期权期限）；A 股常用 SHIBOR 或央行 7 日逆回购利率。
权利金 — Premium / Option Price
期权当前的市场价格，由买方支付给卖方。 = 内在价值 + 时间价值。
权利金 = Intrinsic Value + Time Value
内在价值 — Intrinsic Value
立刻行权能拿到的钱。OTM 与 ATM 期权的内在价值为 0。
Call 内在价值 = max(S − K, 0) Put 内在价值 = max(K − S, 0)
时间价值 — Time Value / Extrinsic Value
权利金中超出内在价值的部分。反映"未来可能更值钱"的不确定性溢价，由 T 与 σ 共同支撑。到期时归零（这就是 Theta 的来源）。
时间价值 = 权利金 − 内在价值
ITM / ATM / OTM — 价值状态 / Moneyness
描述当前现价 S 相对行权价 K 的位置，决定期权是否"实值"。
缩写 中文 Call 条件 Put 条件
ITM实值S > KS < K
ATM平值S ≈ K
OTM虚值S < KS > K

4 波动率相关

波动率是期权交易者真正的"标的"——很多顶级交易员不押方向，只交易波动率。

σ Sigma — 波动率 (年化)
标的资产对数收益率的年化标准差。BS 公式中以小数输入（25% → 0.25）。
典型值：大盘指数 15%~25%；个股 30%~60%；高 Beta 科技股 40%~80%；加密货币 60%+。
σ_年化 = σ_日 · √252 (252 个交易日)
IV — Implied Volatility · 隐含波动率
从市场期权价格反推出的 σ。意为"市场认为未来会有多大波动"。
计算：将 BS 公式倒过来求解（无解析解，需用 Newton-Raphson 等数值方法迭代）。
市场价 = BS(S, K, T, r, IV) → 反解 IV
📌 IV 不是预测值，是市场共识。常常远大于实际后续 HV，这种"波动率风险溢价"是卖方策略的盈利来源。
HV — Historical Volatility · 历史波动率
从过去价格统计出的实际 σ。常用 20 / 30 / 60 日窗口。
HV = √[ (Σ(rᵢ − r̄)²) / (n − 1) ] · √252 其中 rᵢ = ln(Pᵢ/Pᵢ₋₁) 为对数收益率
📊 IV vs HV 的差值是关键信号：IV ≫ HV → 期权"贵"，适合卖方；IV ≪ HV → 期权"便宜"，适合买方。
波动率微笑 / 倾斜 — Volatility Smile / Skew
实际市场上不同 K 的期权 IV 不相同（与 BS 假设矛盾）：
▸ 微笑：远离 ATM 的 OTM/ITM 期权 IV 较高，呈 U 形（FX、商品常见）
▸ 倾斜 (Skew)：股指 OTM Put 的 IV 显著高于 OTM Call（"恐慌溢价"）
反映尾部风险的市场定价。
Vol Crush — 波动率坍塌
重大不确定性事件（财报、FDA、政策）揭晓后，IV 短时间从高位（如 80%）跌回常态（如 25%），导致期权权利金暴跌。即使方向看对，Long Straddle 也可能反向亏损。
⚠ 财报后第二天的 Long 期权头寸通常是巨亏，这是新手最常踩的坑。要么提前锁价（财报前一两天），要么干脆做卖方接收溢价。
VIX — 恐慌指数
芝加哥期权交易所(CBOE)发布的 SPX 30 天 IV 指数，俗称"恐慌指数"。计算基于 SPX 期权的 IV 加权。
典型区间：宁静市 12~16；正常 16~22；紧张 22~30；恐慌 30~50；危机 50+（如 2020 年 3 月达 82）。
IV Rank / IV Percentile
当前 IV 在过去 1 年 IV 区间中的相对位置，用于判断 IV "高"还是"低"。
IV Rank = (当前IV − 1年最低) / (1年最高 − 1年最低) · 100 IV Percentile = 过去1年中 IV 低于当前值的天数比例
📌 实务经验：IV Rank ≥ 50 偏向卖方策略（Iron Condor、Credit Spread）；IV Rank ≤ 20 偏向买方策略（Long Straddle、Long Call）。

5 策略与交易术语

Long / Short — 多头 / 空头
Long (买入)：付出权利金获得"权利"，预期标的或 IV 朝有利方向变化。
Short (卖出)：收取权利金承担"义务"，希望期权到期作废（OTM）赚取时间价值。
Long 与 Short 的盈亏曲线完全镜像。
借方 / 贷方价差 — Debit / Credit Spread
Debit Spread (借方)：净支出，构建时账户减少权利金。如 Bull Call Spread、Bear Put Spread。最大亏损 = 净成本。
Credit Spread (贷方)：净收入，构建时账户增加权利金。如 Bull Put Spread、Bear Call Spread、Iron Condor。最大盈利 = 净收入。
📌 选择规律：IV 高用 Credit（享受 IV 回归 + Theta）；IV 低用 Debit（杠杆参与方向）。
盈亏平衡点 — Break-Even Point
到期日策略盈亏 = 0 时对应的标的价格。单腿 Long Call 平衡点 = K + 权利金；Long Put = K − 权利金。Iron Condor 等多腿策略有两个平衡点。
Covered Call — 备兑开仓
持有 100 股正股 + 卖出 1 张 OTM Call。被行权时按约定价交付正股不会爆仓。
用途：长期持仓基础上增强月度现金流；本质把上行收益的封顶卖了。
总盈亏 = (S − S₀) + 收到的Call权利金 − max(S − K, 0)
Protective Put — 保护性看跌
持有正股 + 买入 OTM Put 作为保险。下行被 Put 兜底，上行不封顶（仅扣除 Put 权利金）。
相当于给持仓买"保险"，权利金即保费。
Cash-Secured Put — 现金担保看跌
卖出 1 张 OTM Put + 在账户中冻结 K × 100 现金。如果被行权，用现金以 K 接货（可视为"打折买股"）。
没被行权则白赚权利金。是 Wheel 策略的第一步。
Wheel 轮策略 — The Wheel
三步循环：① CSP 卖 Put 收权利金；② 若被指派接到正股，转 Covered Call 卖 Call；③ 若 CC 被行权交付正股，回到第①步。
本质是利用愿意持有的标的反复收割权利金，是稳健现金流策略。
Collar 领口策略 — Collar
持有正股 + 买 OTM Put + 卖 OTM Call。Put 提供下行保险，Call 权利金抵消 Put 成本（理想状态零成本）。代价是上行收益被 Call 封顶。
机构对冲长期持仓的经典做法。

6 风险与执行术语

Assignment — 指派
作为 Short 端，对手方行权导致你被强制履约。Short Call 被指派 → 必须以 K 价交付 100 股；Short Put 被指派 → 必须以 K 价买入 100 股。
被指派是卖方的"职业风险"，资金不足时可能被强平。
Exercise — 行权
作为 Long 端，主动履行合约权利。Long Call 行权 → 以 K 价买入 100 股；Long Put 行权 → 以 K 价卖出 100 股。
欧式期权仅到期日能行权；美式期权到期前任意交易日均可。
Early Exercise — 提前行权
美式期权到期前的行权。理论上无股息 Call 不应提前行权（损失时间价值）；但 Put 在深度 ITM、Call 在除息前可能合理。
实务：散户被提前指派的概率极低（< 5%），主要发生在分红除权前夜。
美式 / 欧式期权 — American / European
▸ 美式：可在到期前任意时刻行权。美股个股期权、ETF 期权均为美式。BS 模型仅适用于近似估值（精确需用二叉树或有限差分）。
▸ 欧式：仅在到期日行权。指数期权（SPX、A 股 50ETF）多为欧式。BS 模型解析解严格成立。
Margin — 保证金
卖方为可能履约义务预存的担保金。买方只付权利金、不需保证金。
▸ 裸卖期权：保证金最高（如 SPAN 模型，按潜在最大亏损算）
▸ 价差策略：因风险有限，保证金 ≈ 最大亏损，效率高得多
▸ 备兑/担保：用现金或正股担保，无需额外占用
Pin Risk — 钉住风险
到期日标的收盘价恰好等于行权价 K 时，你不知道自己 ATM 期权会不会被自动行权 / 自动指派。下一交易日开盘出现意外的多头/空头头寸，导致开盘跳空风险。
缓解：到期日临近收盘前主动平仓所有 ATM 期权。
Delta Hedge — Delta 对冲
通过持有标的多空头寸抵消组合 Delta，使总 Δ ≈ 0，从而中性化方向风险，专注捕捉 Vega / Theta / Gamma 收益。做市商的核心工作。
因 Δ 随 S 变化，Delta Hedge 需持续动态调整（这就是 Gamma 风险的根源）。
对冲股数 = −Δ_组合 × 100 (每张期权对应 100 股)
