何时调整 · 何时止损 · 何时持有

关于本指南

"建仓简单，平仓难。" 期权交易 70% 的失败不在选择策略，而在持仓走错时不知道怎么救。本页提供一套结构化的调整决策框架：先用通用三问诊断处境，再按策略类型查找具体的修复方案，最后用真实案例看完整流程。

1 通用决策框架
在做任何调整之前，先回答这三个问题。它们能过滤掉 80% 不必要的"病急乱投医"。

三问诊断法

Q1 · 我的原始论点还成立吗？
建仓时基于的方向 / 波动率 / 事件假设是否仍然有效？如果原始论点已经失效（如财报已揭晓但反应与预期相反），任何"加仓摊低成本"或"对冲拖延"都是在恶化处境，应该止损而不是调整。
Q2 · 当前风险敞口是否还在我的预算内？
建仓时设定了最大可承受亏损（如总资金的 1-3%）。当前未实现亏损是否已超过预算？是否会因 Greeks 变化而进一步放大？风险已超必须果断减仓或对冲。
Q3 · 调整后的新头寸是不是我现在愿意建的？
用"归零再建"思维：假设现在是空仓，给我同样的资金和当前行情，我会建这个调整后的头寸吗？如果不会，那调整本身就是错误的——你只是在用旧仓位绑架自己。
2 Long Call / Long Put 调整
买方策略时间是敌人。调整的核心矛盾：方向看对了但时间不够。

买入期权 Long Call / Long Put · 借方

盈利方向看对，已浮盈 50%+ ⚡ 信号：Δ 已接近 ±0.7-0.8、剩余时间还充裕、IV 上升 ✓ 选 1：分批止盈 — 平掉 50% 锁定利润，剩下 50% 让其奔跑（设移动止损）
选 2：Roll Up — 平掉当前实值期权，买更高 K 的 OTM 期权（"用利润再下注"）
避免："舍不得卖"——剩余 30% 时间价值会被 Theta 蚕食
亏损方向暂时看错，浮亏 30-50% ⚡ 信号：Δ 还很小、Theta 持续吞噬权利金、剩余时间 < 50% ✓ 选 1：止损 — 接受亏损，保留资金机会成本
选 2：Roll Out — 平当前合约，买更远到期 + 同 K（用更多时间换看对方向的概率）
选 3：转为 Spread — 卖一个更高 K 的 Call/更低 K 的 Put 形成价差，回收一些权利金
避免："加仓摊低成本"——双倍下注一个已被市场否决的方向
横盘方向没动，Theta 在咬 ⚡ 信号：每天损失加速、IV 也在下降（Vol Crush） ✓ 果断止损 — Long 期权最忌横盘。买的是"波动 + 方向"两个押注，横盘下两个都输
不推荐：Roll Out（你只是在延后认错）
临近到期剩余 ≤ 7 天 ⚡ 信号：Theta 加速衰减、Gamma 风险放大 ✓ OTM 期权：果断止损（Theta 急速归零）
ATM 期权：除非有强方向催化，否则平仓
ITM 期权：可考虑行权或滚到下月（避免 Pin Risk）
3 Short Put 调整
卖方策略时间是朋友，但方向看错时被指派接货是最大风险。

卖出看跌 Short Put · 含 Cash-Secured Put

盈利方向看对，已收 50%+ 权利金 ⚡ 信号：Δ 已降至 -0.1 ~ 0、距到期还有 >50% 时间 ✓ 提前平仓 — 用 50% 时间赚 50% 权利金，剩余风险/收益比已不划算
资金腾挪 — 平仓后立即开下一个 Put（"50% 利润法则"）
不推荐：持有到期赚最后一分钱——黑天鹅风险
亏损标的下跌，浮亏接近最大亏损 50% ⚡ 信号：Δ 已接近 -0.7、标的接近行权价 K ✓ 选 1：Roll Down and Out — 平当前合约，卖一个更低 K + 更远到期的 Put（继续收权利金，给方向更多时间）
选 2：接受指派 — 如果 K 是你愿意持有的价位，让其行权变成正股，启动 Wheel
选 3：止损 — 如果原始论点失效（如基本面恶化），认错平仓
避免："加倍卖更多 Put"——熊市中是经典送钱方式
临近到期ATM 状态，DTE ≤ 3 天 ⚡ 信号：Pin Risk、Gamma 极大、不知道是否会被指派 ✓ 主动平仓（哪怕只剩 0.05 权利金）— 避免周末跳空指派
或 Roll Out — 滚到下月相同 K，多收 1 倍权利金继续等
4 Vertical Spread 调整
借方价差（Bull Call / Bear Put）和贷方价差（Bull Put / Bear Call）的调整逻辑相反。

借方价差 Bull Call Spread · Bear Put Spread

盈利方向看对，价差已接近最大盈利 80% ✓ 提前平仓锁定 — 剩余 20% 利润需要标的进一步深入实值，风险/收益比恶化
Roll Up/Down（罕见）— 如果方向继续强势，可平掉旧价差换更激进的新价差
亏损方向看错，浮亏 50%+ ✓ 选 1：止损 — 借方价差最大亏损就是净成本，认错最干净
选 2：Roll Out — 平当前价差，用类似净成本买更远到期的同结构价差
避免：把腿"展宽"加大风险——价差被设计成"风险有限"，违反这一点就失去意义

贷方价差 Bull Put · Bear Call Spread

盈利权利金已收回 50-80% ✓ 提前平仓（同 Short Put 50% 法则）— 释放保证金继续滚动
亏损标的逼近 Short 腿 K，Δ 已大幅放大 ✓ 选 1：Roll 到下月同结构 — 多收一倍权利金，给方向修复时间
选 2：转为 Iron Condor — 在反方向开一个贷方价差，把单边风险变成区间风险
选 3：止损 — 当亏损达到 200% 净收入时认错（行业经验值）
5 Iron Condor 调整
铁鹰式有 4 条腿，调整空间最丰富。核心思路：守住未受威胁的一侧，修复受威胁的一侧。

铁鹰式调整 Iron Condor

盈利标的稳在区间中央，已赚 50% 净收入 ✓ 主动平仓（"50% Profit Take"）— 这是 Tom Sosnoff 等机构的标准做法。剩余 50% 利润不值得继续承担尾部风险
亏损单边受威胁（如 Call 那侧） ⚡ 信号：标的接近 K₃（Short Call）、Δ_short_call > 0.3 ✓ 选 1：Roll Up 受威胁的一侧 — 把 Bull Put Spread 那侧整体上移到接近现价（多收权利金、扩大上行盈利区）
选 2：Roll Out 整体到下月 — 同结构滚到下个月，多收一倍权利金
选 3：变形为蝶式 — 卖出 Call 价差移近，买出 Call 翼也跟着移，构造 Reverse Iron Butterfly
避免：只调整未受威胁的一侧——锁死的是机会，不是风险
亏损双边受威胁（罕见，标的剧烈波动后回归） ✓ 整体止损 — 预期已被市场证伪，再调整只是延后认错
临近到期受威胁腿在 ATM 附近 ✓ 必须主动平仓 / 滚动 — Gamma 风险极大，到期日跳空可能瞬间击穿翼
6 Straddle / Strangle 调整
跨式与宽跨式调整的核心：方向已经明朗时非对称化头寸。

买入跨式调整 Long Straddle

盈利方向爆发后，一侧已大幅盈利 ✓ 选 1：平掉盈利腿，留亏损腿 — 用盈利锁定，亏损腿成"免费看涨权"
选 2：Roll 亏损腿 — 平掉过期的虚值腿，用回收资金调整方向
选 3：双腿全平 — 最干净的做法，避免 Theta 继续衰减
亏损事件揭晓后 IV Crush，双腿同步贬值 ✓ 果断止损 — Long Straddle 最大敌人，认错越早越好。这是事件驱动失败的标准结局

卖出跨式 / 宽跨式调整 Short Straddle / Strangle

亏损单边突破，标的远离 K ✓ 选 1：Roll 受威胁腿 — 把虚值的腿移到更接近现价（多收权利金、降低受冲击侧）
选 2：转为 Iron Condor — 加买保护翼把无限风险变成有限
选 3：变成 Strangle（仅 Straddle 时）— 把双 K 拉开降低 ATM 暴露
关键：当亏损达到收到权利金的 200% 时必须止损
7 Rolling 通用技巧
"展期" 是期权调整最常用的工具。三种主要 Rolling 方式 + 实操规则。
类型操作目的典型场景注意
Roll Out同 K，更远到期给方向更多时间修复Long 期权浮亏、Short 期权临到期 ATM必须收净权利金或零成本，否则等于继续追加投入
Roll Up / Down同到期，调整 K追逐方向（Long）或防守（Short）Long Call 已 ITM 想止盈再下注；Short Put 受威胁Roll Up Long Call 等于"用利润再下注"，要确认论点仍成立
Roll Out and Up/Down同时调整到期 + K双重防守Iron Condor 受威胁的一侧、贷方价差被穿透多腿调整手续费高，确认确实有改善才执行
Diagonal Roll跨腿展期将单腿期权变成日历价差Long Call 浮亏想保留方向但减成本构造复杂，需要熟悉日历价差
📐 Rolling 的黄金准则 · 规则 1：必须收净权利金（贷方）或零成本。如果 Roll 反而要付钱，本质上是加仓，应该重新评估。
规则 2：不要在大幅亏损时 Roll Out 太远——那是给市场更多时间证明你错了。Roll Out 一般以 30-45 天为限。
规则 3：原始论点已经失效时，不要 Roll，应该止损。Rolling 是给"对的策略一些时间"，不是给"错的方向找借口"。
8 止损 vs 调整 vs 持有
最核心也最难的判断。这个矩阵帮助快速决策。
状态组合原始论点风险敞口剩余时间推荐动作
深度浮亏已失效已超预算—立即止损（情绪上最难，但理性上唯一选择）
深度浮亏仍成立仍在预算内≥ 50%Roll Out（给方向时间）
中等浮亏仍成立仍在预算内—持有（小亏不需要调整）
基本横盘——≤ 30%提前平仓（Theta 加速期，剩余收益有限）
中等浮盈仍成立——持有 + 移动止损
深度浮盈———提前平仓（50% 法则、80% 法则）
9 实战案例
3 个真实情境，看完整调整流程。
📚 案例 1 — Iron Condor 单边受威胁的标准修复
①初始：SPY=450 时建 IC：Buy 430P / Sell 440P / Sell 460C / Buy 470C，DTE 45 天，净收 $1.50
②第 20 天：SPY 涨到 458，Short Call 那侧 Δ=0.32，价差已浮亏。检查三问：方向论点（震荡）受挑战但未完全失效；风险敞口仍在预算；剩余时间 25 天。
③调整：Roll Up 整个 IC 到 SPY=458 中心：Buy 438P / Sell 448P / Sell 468C / Buy 478C，多收净权利金 $0.40。
④结果：SPY 回到 455 横盘，新 IC 在区间中央到期作废，总收益 $1.90 × 100 = $190。
✅ 关键：调整后是"零成本或贷方"，且 Roll 后论点更合理（区间向上挪了 8 元）。
📚 案例 2 — Long Call 财报后认错
①初始：NVDA 财报前 1 周，IV=70%，买 ATM Long Call DTE=14，权利金 $8
②财报次日：NVDA 仅涨 2%（市场预期 +5%），IV 暴跌至 40%（Vol Crush）。Call 价值 $4.5，浮亏 44%。
③三问诊断：原始论点（财报超预期 + IV 上升）已完全失效。继续持有等于赌"还能涨回来"——新的赌博，不是原计划。
④动作：当天平仓止损，亏损 $3.5。
✅ 关键：原始论点失效时，立即止损。如果当时 Roll Out 到下月，会被 Theta 继续吞噬，到期前可能归零。
📚 案例 3 — Short Put 接货转 Wheel
①初始：看好 AAPL 长期，180 价位 CSP，K=170，DTE 30，收 $2.50
②到期前 5 天：AAPL 跌到 168，Short Put 已浮亏 $1。Δ_put=-0.65。
③三问诊断：原始论点（长期看好）成立；K=170 是愿意接货的价位；不需要调整。
④动作：持有到期，被指派以 $170 接 100 股 AAPL。实际成本 = 170 - 2.50 = $167.50（含权利金）。
⑤下一步：启动 Wheel — 卖 1 张 OTM Covered Call @ K=180，再收一份权利金。如此循环。
✅ 关键：被指派不是失败。如果 K 是你愿意持有的价位，接货就是计划的一部分。

📌 风险提示：调整不是万能药，更不是"避免认错"的工具。最专业的交易者也会有 30-40% 的策略亏损，关键是每次亏损都不要超过预算。
